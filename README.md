# Django Lab

## Installation

- Python 3.10.6

- Pip  

	``` 

	python3 -m pip install --upgrade pip 

	```
- VirtualEnvironment

	```

	pip install virtualenv

	```

- Create and Activate VirtualEnvironment

	```

	virtualenv name

	```

	```

	source name/bin/activate

	```
- 
- Run server in devops directory
	```

	python3 manage.py runserver

	```
